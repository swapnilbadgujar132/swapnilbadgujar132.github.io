TezChal
● Live Product Mobile + Web Backend Grocery Delivery
Visit tezchal.com →

A complete backend system for an instant grocery delivery platform inspired by Zepto and Blinkit. TezChal manages the full order lifecycle — from customer placement to real-time tracking to doorstep delivery — with smart delivery partner assignment and live inventory control. The RESTful APIs power both mobile apps and the web frontend, with Laravel Sanctum for secure authentication and role-based access for all user types.

Real-Time Order Processing — instant placement, live status tracking, order management
Smart Delivery Partner Assignment — auto-assign nearest available delivery agent
Live Inventory Management — stock tracking, low-stock alerts, product availability sync
Secure Authentication via Laravel Sanctum — token-based API auth for mobile & web
Razorpay Payment Integration — online payments, refund handling, payment tracking
Role-Based Modules — Admin, Customer, and Delivery Partner with separate dashboards
Optimized RESTful APIs — designed and consumed by both mobile app and web frontend
